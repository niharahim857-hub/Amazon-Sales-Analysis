# Amazon Sales Analysis Dashboard (Power BI)

#Project Overview
This project analyzes Amazon sales data to uncover key business insights using Power BI. The dashboard provides a clear view of sales performance, profit trends, and customer behavior.

# Objective
To analyze sales data and identify patterns, trends, and key factors affecting business performance.

# Tools & Technologies
- Power BI  
- Excel / CSV Dataset  
- DAX (Data Analysis Expressions)  

# Key Features
- Interactive dashboard with filters and slicers  
- Sales analysis by region, category, and segment  
- Profit and revenue trends visualization  
- Top-performing products identification  
- Customer segment analysis  

# Key Insights
- Sales are highest in specific regions and product categories  
- Certain products contribute more to overall profit  
- Seasonal trends impact sales performance  
- Discounts can influence profit margins
